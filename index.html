CSS Grid

C SS Grid Layout — це двовимірна система макетування, яка дозволяє розміщувати
елементи в рядках і стовпцях. Grid є найпотужнішим інструментом для створення макетів у CSS, що забезпечує можливість точного контролю над розташуванням елементів на сторінці.

На відміну від Flexbox, який орієнтований на одновимірні макети, Grid дозволяє контролювати обидва виміри одночасно. Це означає, що ви можете визначити як рядки, так і стовпці, а потім розмістити елементи в будь-якій комірці цієї сітки. Ця можливість робить Grid ідеальним для створення складних і точних макетів.

Історія

CSS Grid Layout був розроблений для вирішення складних проблем макетування, які існували в CSS протягом багатьох років. Перші ідеї Grid з'явилися ще в 2007 році, але офіційна специфікація почала розроблятися лише в 2010 році. Перша публічна робоча чернетка була опублікована в 2012 році, а в 2017 році Grid став доступним у всіх основних браузерах. Це стало революційним моментом у веб-розробці, оскільки дозволило створювати складні макети без використання хаків і обхідних шляхів.

Термінологія

Ось кілька ключових термінів, пов'язаних з CSS Grid:

Grid Container — елемент, до якого застосовується властивість display: grid. Він стає контейнером для всіх дочірніх елементів Grid.

Grid Container — прямий нащадок Grid Container. Ці елементи розміщуються в сітці відповідно до правил, визначених для Grid Container.

Grid Line — горизонтальні та вертикальні лінії, які розділяють сітку на рядки та стовпці. Вони можуть бути пронумеровані або названі.

Grid Track — простір між двома сусідніми лініями сітки. Це може бути рядок або стовпець.

Grid Cell — перетин рядка та стовпця в сітці. Це найменша одиниця в Grid, яка може містити контент.

Grid Area — будь-який прямокутний простір, обмежений чотирма лініями сітки. Він може охоплювати кілька рядків і стовпців.

Основні властивості

Grid має багато властивостей для контролю макета. Ось деякі з найважливіших:

grid-template-columns — визначає кількість і розмір стовпців у сітці.

grid-template-rows — визначає кількість і розмір рядків у сітці. grid-gap — встановлює відстань між рядками та стовпцями.

grid-template-areas — дозволяє називати області сітки та розміщувати елементи в цих областях.

justify-items — вирівнює елементи по горизонталі всередині їхніх комірок.

align-items — вирівнює елементи по вертикалі всередині їхніх комірок.
